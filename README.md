# ❗ 주의 ❗
---
- 현재 terraform 하실 때 `terraform apply` 를 여러 사람이 하면 충돌이 나는 것 같습니다
- `terraform apply` 는 박세진님이 하고 있는 상황이고,
- 본 레포지토리가 수정될 때 세진님이 `terraform apply` 를 할 예정입니다
- 그러니 terraform 레포지토리 내용에 **수정 사항이 생겨 새로 `terraform apply` 를 해야 한다면 수정사항 pr 올려주시고 세진님께 말씀해주세요!**
- pr 머지하고 세진님이 `terraform apply` 할 예정입니다
